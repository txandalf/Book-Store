#### SMALL BOOK STORE ####

# La idea es crear una aplicación web con Python
# para familiarizarme cada vez un poco más con los
# conceptos del desarrollo web.

# Usando Flask como microframework web, jinja2 para la
# generación de plantillas html, sqlalchemy para abstraernos
# de la operaciones SQL independientemente del motor de bases
# datos que usemos, y sqlite3 como motor de base de datos (un fichero
de almacenamiento local)

### COMANDOS ###

>>>python manage.py drop_tables

>>>python manage.py create_tables

>>>python manage.py add_data_tables

>>>python manage.py runserver
